# Amazon Sales Dashboard

An interactive Power BI dashboard built using Amazon product sales data from Kaggle. The dashboard provides insights into product ratings, reviews, discounts, brands, and category performance through easy-to-understand visualizations.

---

## Project Overview

This project focuses on analyzing Amazon product data using **Microsoft Excel** and **Power BI**. The dashboard helps identify top-performing products, customer rating trends, discount distribution, and brand performance through interactive visuals.

---

## Dataset

The dataset is from kaggle,stored in an **Excel (.xlsx)** file and contains information such as:

- Product Name
- Brand
- Category
- Product Rating
- Total Reviews
- Discount Percentage
- Product Price

## Dashboard Features

The dashboard includes the following insights:

### KPI Cards
- Average Rating
- Total Products
- Average Discount
- Total Categories

### Visualizations
- Top 15 Products by Total Reviews
- Top 10 Brands by Reviews and Average Rating
- Total Products by Rating Category
- Total Products by Discount Band
- Good Product Ratio Gauge
- Category Filter (Slicer)

## Key Insights

- Products with higher review counts generally maintain strong customer ratings.
- Most products fall within the **0–50% discount** range.
- The dashboard highlights the top-performing brands based on reviews and ratings.
- Category filtering enables focused analysis of specific product groups.
- KPI cards provide a quick overview of overall business performance.

4. Modify or refresh the Excel dataset if needed.
